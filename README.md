# 🚗 Sri Lanka Vehicle Valuation & RAG Compliance Agent

An intelligent Agentic RAG system built to assist with real-world SME vehicle valuation, import tax calculation, and policy compliance in Sri Lanka.

---

## 🌟 Key Features
- **Automated Valuation:** Intelligent price estimation based on vehicle datasets.
- **RAG Policy Compliance:** Seamless document querying for Sri Lankan import duties, luxury tax policies, and valuation guidelines.
- **Fast AI Inference:** Powered by Groq's low-latency LLMs and ChromaDB vector indexing.
- **Interactive UI:** Clean Streamlit dashboard for effortless querying and setup.

---

## 🛠️ Tech Stack
- **Framework:** Streamlit
- **AI / RAG:** LangChain, ChromaDB, Groq API
- **Data:** Pandas, Custom CSV & Policy Documents
